# Linkapp
## Directorys And Files
* app/ 
    * /base.html
    * page/
    * style/ (scss)
    * script/ 
* build/
    * /all_pages.html
    * stlye/ (compiled css)
    * script/

## Atributos Link
* link
    - Linka um elemento base com seu elemento conteúdo.
* link-remove
    - Caso exista um link-remove de valor "true" no elemento base ou elemento conteúdo, esse não é gerado.
